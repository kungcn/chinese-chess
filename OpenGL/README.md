Build
-------------

You should install **GLEW** **GLFW3** **SOIL** **ASSIMP** first.
Here are the link:

>- [GLEW][1]
>- [GLFW3][2]
>- [SOIL][3]
>- [ASSIMP][4]

Then in the Computer-Graphic-Project folder

```
cmake .
make
```


----------

Run
-------------


```
./CG_Project
```
